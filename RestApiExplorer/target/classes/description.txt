Реализовано web приложение для доступа к файловой директории сервера с помощью Restful api, Spring Framework и Jquery.
Для запуска используется spring-boot-starter-thymeleaf, подключенный с помощью сборки maven.

ExplorerController реализует Rest api логику и создает три страницы /filetree - главная страница берется из шаблона
filetree.html и выводится в формате html,
/treedata - json страница с информацией о файловой системе,
/file - json фтраница с информацией об открытом файле.

DataBuilder отвечает за формирование денных - лист из элементов файлового дерева - TreePoint,
считывание файла в FileData.

TreePoint содержит пусть к элементу, его тип и его название.
FileData содержит название файла и его содржимое.

В ExplorerController создается view с параметром path, которая запрашивает лист у DataBuilder.
Так же формируется и страница с информацией о выбранном файле (только вместо листа один объект FileData).

На главной странице /filetree выводится первоначалоное содержание корневой директории в левой части страницы,
пустое поле для дальнейшего открытия файлов - в правой.
Главная view реализует метод onClick для элементов файлового дерева openDirOrFile().
В методе проверяется какому типу принадлежит объект. Если тип dir, то посылается запрос на страницу
localhost:8080/treedata?path=<путь к выбранной директорие>.
Так основная страница получает информацию о простраивает дерево дальше.
Если тип file, то посылается запрос на страницу localhost:8080/file?path=<путь к выбранному файлу>,
далее выводится название файла и его содержание в правое поле.
Возможно чтение файлов с расширением .txt, .xml, .html, .css, .js, .json и тд.

Реализована сборка с помощью maven. Возможно собрать jar и запустить (spring-boot-starter-thymeleaf сам запустит сервер).
